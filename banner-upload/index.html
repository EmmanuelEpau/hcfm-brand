First-time setup — one-time

To upload banners, you need a personal access token from GitHub. This is a long string that proves you have permission to update the brand repo. Get yours from Emmy or generate one yourself.

  1. Open github.com/settings/personal-access-tokens/new in a new tab.
  2. Token name: HCFM Banner Studio. Expiration: 90 days (or your team policy).
  3. Repository access: Only select repositories → EmmanuelEpau/hcfm-brand.
  4. Repository permissions → Contents: Read and write. Leave everything else as default.
  5. Click Generate token. Copy the token (starts with github_pat_) and paste it below.

Your token is stored only in this browser (localStorage). It's never sent anywhere except GitHub's own API. Clear it any time from the Settings panel below.
